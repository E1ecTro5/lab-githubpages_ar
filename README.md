# Лабораторная работа №2: "Публикация статической страницы через GitHub Pages"
Работа представляет из себя статическую страницу, развернутую с помощью сервиса GitHub Pages.
> [!WARNING]
> Страница оптимизирована (разработана) исключительно под разрешение 1920х1080. Для других разрешений тут не предусмотрены уступки.

Ссылка на сайт указана в описании репозитория. Ну или просто можете [нажать сюда](https://e1ectro5.github.io/lab-githubpages_ar/).
